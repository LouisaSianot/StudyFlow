🚀 Automated My Entire Study Workflow — Here's How

I spent 10-15 hours every week just organising study materials, downloading files from Google Classroom, creating notes, and tracking assignments. As a student, that's time I needed for actual studying!

So I built an automation system that does all of this for me. Here's what it does:

✅ Monitors Google Classroom 24/7 for new materials ✅ Automatically downloads and organises every file locally ✅ Uses Google Gemini AI to generate: • Comprehensive summaries • Review questions with answers • Flashcards for key concepts ✅ Creates interconnected notes in Obsidian ✅ Tracks every assignment and deadline automatically ✅ Records grades and analyses performance trends

🎯 The Results: • Saved 12+ hours per week • Never missed a single deadline this semester • Improved my GPA by staying consistently organised • Built a searchable knowledge base of 200+ interconnected notes • Reduced study stress significantly

🛠️ The Tech Stack: • n8n for workflow automation • Google Classroom API for material retrieval • Google Gemini AI API for content generation • Obsidian for knowledge management • Docker for self-hosting n8n

💡 Key Learnings:

1. Automation isn't just for software engineers — students can benefit immensely
2. AI can enhance learning, not replace it (Gemini generates study aids, I do the studying)
3. Proper knowledge management makes a huge difference in academic performance
4. Small time investments in automation pay massive dividends
5. Open-source tools can create enterprise-level solutions

💰 Costs: Free tier on Gemini AI covers most use cases ⏱️ Setup Time: ~10 hours initial, then basically zero maintenance
